Aclarando conceptos

¿Que es HTML 5?

El “Hypertext Markup Language” mas Conocido es un Lenguaje de programacion que hace referencia a todo lo que es la Web.

¿Que es Responsive?

El diseño web responsive o adaptativo. es de las mejores tecnicas del diseño Web, que busca la visualizacion correcta de una pagina Web ya sea en un Dispositivo Tables, y moviles

¿Que es CSS 3?

Mientras que HTML nos permite una estructura de pagina Web, las Hojas de estilo en Cascada (Cascading Style Sheets o CSS) son aquellas reglas que permiten una mejor visualizacion de tu pagina Web
